# Sachin Chelaka Fernando Digital Resume

Static digital resume built as a polished operations dashboard for business
analysis, logistics flow, KPI reporting, CRM-backed service work, and analytics
projects.

## Files

- `index.html` - resume content and page structure
- `styles.css` - responsive Apple-inspired dashboard styling
- `script.js` - print action, email copy action, scroll progress, and active nav

## Local Preview

Open `index.html` directly in a browser, or run:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.

## GitHub Pages

This repository is ready for GitHub Pages. Publish the site from the `main`
branch root. No build step is required.
